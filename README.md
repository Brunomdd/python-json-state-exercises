#Exercícios de Persístencias  e Gerenciamento de  Estado em Python
Esse repositorio contém exercícos práticos focados em:
- Gerenciamento de estado na memória (RAM)
- Persistencia de dados com JSON
- Validações e regras de negócio
- "Separação entre lógica e interface
- Todos os exercícios foram desenvolvidos com foco em engenharia de sotfware e no entedimento de como sistemas lidam com mudanças de estado.
